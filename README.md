# Verify
Herramienta en bash para verificar si los sitios de la Red Tor .onion estan activados o desactivados antes de ingresar a un sitio en la dark weeb

Instalar 

git clone https://github.com/AndJ0x8/Verify

cd Verify/

chmod +x setup.sh

bash setup.sh

systemctl start tor

chmod +x verify.sh

bash verify.sh
